# Drasi Test Parseable

This repository contains various YAML configuration files, likely for a testing, monitoring, or deployment setup.

## Files:

- `continuous-query.yaml`: Configuration for continuous data queries.
- `debug-reaction.yaml`: Defines reactions or responses for debugging events.
- `explore-query.yaml`: Configuration for exploratory data queries.
- `http-reaction.yaml`: Defines reactions or responses for HTTP events.
- `k8s-source.yaml`: Kubernetes source configuration.
- `kind-kubeconfig.yaml`: Kubernetes configuration file for Kind (Kubernetes in Docker).
- `nginx.yaml`: NGINX web server configuration.
- `result-reaction.yaml`: Defines reactions or responses based on query results.
